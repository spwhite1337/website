# Scott's Website

Personal Website with flask+vue to display data products and my blog.

## Procedure

- Python 3.5
- Node.js
- npm
- aws credentials


## Set up Client-Side

- `cd frontend`
- `npm install`
- `npm run dev`

## Set up Server-Side
- `cd backend`
- `export / set FLASK_APP=run.py`
- `export / set FLASK_DEBUG=1`
